Graphing libraries and tools written in TypeScript.

Libraries

graph-lib

Serving as the backbone for all of the tools and libraries in the graph-ts ecosystem, graph-lib provides the data structure and functional interface used to represent and manipulate graphs. It is designed to play nicely with libraries like Immer, allowing developers easily work with immutable state (but not requiring it).

graph-react

The graph-react library is a customizable React component for rendering and interactively manipulating graphs.
