Graphing libraries and tools written in TypeScript.

Libraries

These are the libraries of graph-ts. Behold them in all their glory!

graph-lib

This is a description of the graph-lib library. It talks about things like how the library is designed to work with state-based libraries like immutable.js and the like. I'll fill this all in later when I have time to sit down and think about all that needs to be said.

graph-react

This is a description of the graph-react library. It talks about all of the amazig functionality that you get out of the box. Things like interactivity, undo and redo, selections, and more! It's really a pretty amazing React component and you should definitely check it out. I think I'll even include a screenshot below this, or maybe a super-duper simple demo.
