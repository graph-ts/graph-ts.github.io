## Basic Examples

The following examples give a very basic overview of what it's like to work with
the various ```graph-ts``` libraries.

### Create and search a graph
The ```graph-lib``` library is used to create, manipulate, and work with directed graphs.
In this example, we create a small graph with 5 nodes and 5 edges and then use breadth-first
search to find a specific node.

<TabbedCodeBlock>

```typescript
import { newGraph, breadthFirst, Graph, Node, Edge } from '@graph-ts/graph-lib';

// Create an array of graph nodes
const nodes: Node[] = [
    { id: 'a' },
    { id: 'b' },
    { id: 'c' },
    { id: 'd' },
    { id: 'e' }
];

// Create an array of graph edges
const edges: Edge[] = [
    { id: 'ab', source: 'a', target: 'b' },
    { id: 'ac', source: 'a', target: 'c' },
    { id: 'bd', source: 'b', target: 'd' },
    { id: 'be', source: 'b', target: 'e' },
    { id: 'ce', source: 'c', target: 'e' }
];

// Create the graph
const graph: Graph = newGraph(nodes, edges);

// Starting at node 'a', search for node 'd' using breadth-first search
breadthFirst(graph, 'a', (node: Node) => {
    if (node.id === 'd') {
        console.log('Found it!');
        return true; // return true to stop traversal
    }
});
```

### Create and visualize a graph

The ```graph-svg``` library provides a [React](https://reactjs.org/) component for rendering
graphs built using the ```graph-lib``` library. In this example, we lay out the graph
above by adding x- and y- coordinates to the nodes and passing it to a ```<GraphSVGDiv>```
component.

<TabbedCodeBlock>

```tsx
import { newGraph, Graph, Node, Edge } from '@graph-ts/graph-lib';
import { GraphSVGDiv, Vector2, ShapeDef } from '@graph-ts/graph-svg';

// Create an array of graph nodes with x- and y-coordinates
const nodes: Node<Vector2>[] = [
    { id: 'a', x: -100, y:  0  },
    { id: 'b', x:  0,   y: -60 },
    { id: 'c', x:  0,   y:  60 },
    { id: 'd', x:  100, y: -60 },
    { id: 'e', x:  100, y:  60 }
];

// Create an array of graph edges
const edges: Edge[] = [
    { id: 'ab', source: 'a', target: 'b' },
    { id: 'ac', source: 'a', target: 'c' },
    { id: 'bd', source: 'b', target: 'd' },
    { id: 'be', source: 'b', target: 'e' },
    { id: 'ce', source: 'c', target: 'e' }
];

// Create the graph
const graph: Graph<Vector2> = newGraph(nodes, edges);

// Create a default node shape
const defaultShape: ShapeDef = {
    shape: 'circle',
    radius: 15
}

// Rendered below
<GraphSVGDiv
    graph={graph}
    defaultShape={defaultShape}/>
```
